| 快捷键   | 作用                                     |
| -------- | ---------------------------------------- |
| 0        | 数字0，移到光标所在行的行首              |
| $        | 移到光标所在行的行尾                     |
| gg       | 移到文首                                 |
| GG       | 移到文尾                                 |
| /word    | 向下查询字符串word                       |
| ?word    | 向上查询字符串word                       |
| n        | 继续向下查找字符串                       |
| N        | 继续向上查找字符串                       |
| dd       | 剪切光标所在行                           |
| ndd      | 剪切从光标所在行开始向下的n行            |
| yy       | 复制光标所在行                           |
| nyy      | 复制从光标所在行开始向下的n行            |
| p        | 将已复制或者剪切的数据在光标的下一行粘贴 |
| u        | 撤销当前操作                             |
| :line_no | 跳到第line_no行                          |
| set nu   | 显示行号                                 |
